1) Создайте функцию, которая выводит в консоль определенное количество сообщений "Hello world". Используйте цикл.
Пример: showText(5) выводит в консоль 5 раз текст "Hello World"

2) Модифицируйте функцию showText чтобы она принимала ДВА аргумента: количество раз, само сообщение
Пример: showText(5, "Hello world") выводит в консоль 5 раз текст "Hello World"

3) Модифицируйте функцию showText чтобы она принимала ЧЕТЫРИ аргумента: количество раз, само сообщение, символы с начала строки, символы с конца строки.
Пример: showText(5, "Hello world", "<<<", ">>>") выводит в консоль 5 раз текст "<<< Hello World >>>"

Переместить нули в конец массива

Есть массив:
[7, 2, 3, 0, 4, 6, 0, 0, 13, 0, 78, 0, 0, 19, 14]

Нужно получить:
[7, 2, 3, 4, 6, 13, 78, 19, 14, 0, 0, 0, 0, 0, 0]

Но нельзя использовать методы и функции массивов. Никаких pop, push, filter, map и т.д.
